The program is in Java and i have compiled everything as an executable jar file.

To run the jar file please use following command:

$ java -jar RubineFeatures.jar <pathtolettersdata>

INPUT : My program expects the input as provided in the directory structure of data for letters, Also it takes the naming of the files in the same way

OUTPUT : CSV file consisting of all the labels and the data for 13 rubine features.

SAMPLE:

$ java -jar RubineFeatures.jar D:\Education\MSCS\Semester-I\Courses\Sketch_Recognition\HomeWork\HW1\Data\letters-txt

$ java -jar RubineFeatures.jar ./Data/letters-txt/
